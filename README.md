#### Get started for development:

Ensure `bun` is installed
```sh
bun --upgrade
```

↑ if that fails:

```sh
curl -fsSL <https://bun.sh/install> | bash
# or npm install --global bun@latest
```

```sh
bun --version
```
↑ should work now
```sh
bun install
```

```sh
bun run dev
```
open browser at http://0.0.0.0:3033
